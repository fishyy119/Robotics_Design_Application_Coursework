# hw25 MPC

本目录中的正式作业代码为 `hw25_closed_loop_mpc.py`。

## 内容说明

- 保留参考代码中的线性倒立摆 `LIPM`、离散预瞄模型、`CoP/ZMP` 支撑域约束和二次规划求解形式。
- 实现开环 MPC：仅根据初始时刻的检测状态求解一次完整 `CoP` 序列。
- 实现闭环 MPC：每个采样时刻根据当前检测到的 `CoM` 状态重新构造截断时域并滚动优化，只执行当前步的第一个控制量。
- 对比开环与闭环在状态检测误差下的鲁棒性。

## 扰动设置

- 实际系统初值在名义初值基础上加入微小状态偏差。
- 在递推过程中，对状态更新加入很小的附加扰动，模拟模型误差或外界扰动。
- 开环 MPC 仅在 `t = 0` 基于当前状态求解一次，然后整段执行该 `CoP` 序列。
- 闭环 MPC 在每个采样时刻都读取当前 `CoM` 状态，并基于剩余时域重新求解。
- 默认配置下未额外加入测量噪声；如需模拟传感器噪声，可直接调整脚本中的 `measurement_noise_std`。

## 运行方式

依赖：

- `numpy`
- `scipy`
- `matplotlib`
- `quadprog`
